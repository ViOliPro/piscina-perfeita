# Piscina Perfeita
 Aplicacao para funcionarios ou empresas que trabalham com limpeza e venda de produtos para piscinas.

 ## Sobre o Projeto
 O projeto tem como objetivo facilitar a vida de quem trabalha com piscinas, seja na limpeza ou na venda de produtos. A aplicação oferece uma interface intuitiva e fácil de usar, permitindo que os usuários possam gerenciar suas tarefas diárias de forma a organizar e acompanhar o "bem estar" da piscina e a quantidade de produtos usados, bem como ajudar a ter um historico da quantidade de materiais utilizados em um determinado período.

 ## Tecnologias Utilizadas
	- PostgreSQL
	- API .Net
	- JWT
	- React WEB
	- React Native
